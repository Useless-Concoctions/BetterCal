# BetterCal

A premium, local-first intelligent calendar designed to bridge the gap between static scheduling and the fluid reality of daily life.

## Problem

The modern calendar is a rigid container. Life is liquid. When a single meeting runs over, the rest of your day "breaks," leading to scheduling debt, manual dragging, and eventually, abandonment of the plan. BetterCal turns your schedule into a fluid system that adapts in real-time.

## Features

- **Liquid Scheduling**: An elastic engine that automatically reschedules "Ghost Goals" around life's commitments. If you're late, your goals move with you.
- **Weather-Aware Context**: Automatic detection of precipitation and temperature extremes for `#outdoor` goals.
- **Semantic Intuition**: Inferred energy windows (e.g., Deep Work in the morning, Errands in the afternoon) without manual setup.
- **Advanced Goals**: Define objectives with complex frequencies (e.g., "5 times a week") and explicit durations.
- **Zero-Friction NLP**: A superior natural language parser that understands intent, duration, and frequency without complex menus.
- **Universal Drag-and-Drop**: Buttery smooth interactivity across Month, Week, and Day views with optimistic UI reactivity.
- **Intelligent Suggestions**: Quick-start goal blueprints directly in the Command Bar to minimize input friction and kickstart productivity.

## Stack

- **AI**: Next-generation NLP via `chrono-node` and custom heuristic engines.
- **Frontend**: React 19, Next.js 15, Framer Motion, Lucide.
- **Database**: Local-First SQLite via Prisma 7 and LibSQL.
- **Desktop**: Tauri for a native, high-performance experience.
